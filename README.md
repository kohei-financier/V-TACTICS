## 🟡 サービス概要

本サービスは、YoutubeやXなどに散在するValorant（ヴァロラント＜FPSゲーム＞）のテクニック情報を、効率的に検索・情報管理できるツールです。


検索したいキャラクター名などでテクニックをYoutubeとXを横断して検索できる機能や、ユーザーごとにブックマークやカテゴリ分けできる機能を実装し、
ユーザーが知りたいと思ったテクニックにすぐにアクセスできる環境を提供します。

## 💭 このサービスへの思い・作りたい理由

キッカケは、友人とアマチュアを目指してチームを組んでいた（Premier）時に、

**当時使っていたDiscordではチャットで情報が埋もれてしまい、過去の良かったテクニック動画が見つけられないこと**でした。

また、VALORANTには、エイムの練習法やキャラクターのスキルの使い方など、多くのテクニック情報が存在します。

しかし、これらの**テクニック情報はYouTubeやXに大量に点在しており、チームにヒアリングしたところ、「プロが使っているテクニックを調べたくても、なかなかすぐにたどり着けない」という悩みがある**ことが分かりました。


そこで、<ins>知りたいテクニックを「素早く」「簡単に」引き出せるサービスを開発したい</ins>と考え、着手しました。


## 🙋 このサービスのポイント

#### 1️⃣ <ins>🔍️YoutubeとXを横断検索！散らばっている情報源が「V-TACTICS」なら一括検索！</ins>
【使用技術：Ransack, Autocomplete】

VALORANTのテクニックはYoutubeとXに投稿されていることが多いですが、プラットフォームが違うためすぐ比較ができません。  
「V-TACTICS」なら100件を超えるテクニックを掲載。困ったら、まずはこのサイトで検索！

#### 2️⃣ <ins>📊人気テクニックがすぐわかる！初心者やサイトを始めて知った人向けに、お気に入り順表示機能搭載！</ins>
【使用技術：Swiper, Turbo Frames】

「V-TACTICS」なら、Google検索では難しい「人気なテクニックをすぐ知りたい！」を実現！短いローディング中でもすぐ調べられます。

#### 3️⃣ <ins>⭐️ログインで、動画投稿・マイページ作成が可能！VALORANTのテクニックなら本サイト！</ins>
【使用技術：Stimulus, YoutubeAPI, Turbo Stream】


> [!NOTE]
> 掲載方法はリンク埋め込みであるため、動画再生数やインプレッション、著作権などは投稿主に還元されます。当方に収益はございません。  
> また、違法なサイトへのリンクが本サービス内に見られる場合は、管理主が対処いたしますので[お問い合わせフォーム](https://forms.gle/3SfRcjCQLi6dcsL18)よりご連絡ください。




## 👨‍👨‍👧‍👧 本サービスのターゲット

#### 1. プロやアマチュアプレイヤー
  - 上達を目指して、プロのテクニックや最新のメタなどを積極的に学びたい
  - テクニックをよく収集していて、効率よく検索・管理できる方法を模索している
#### 2. 初心者・新規プレイヤー
  - Valorantを始めたばかり or これからValorantを始めるユーザーで、基礎的なテクニックを調べたい
  - 周りに詳しい友人もおらず、どこから情報を得てよいのか分からない

## 🔭 本サービスの狙い・将来的なビジョン

- YoutubeやXを横断で検索できることで、今まで知らなかったテクニックのテクニック動画制作者の認知向上。
- 素早く知りたい情報にアクセスできることで、新規プレイヤーのゲームへの心理的ハードルも下げる。
- VALORANTのユーザー数向上。eスポーツへの盛り上がりへ貢献。
- サービスのフォーマットを横展開し、料理やスポーツなど、ゲームに限らず動画情報収集の基盤に。

## 🌐 実装機能一覧

**【MVPリリース】**
- テクニックの投稿機能
- テクニックの一覧機能
- テクニックの検索機能
- お気に入り機能
- ユーザー登録・ログイン機能
- i18n

**【本リリース】**
- ユーザー登録・ログイン機能（Google認証）
- お気に入りのカスタマイズ化（自身がつけたフォルダで管理できる）
- シェア機能・OGP機能
- カテゴリーのフォロー機能
- 投稿時のプレビュー画面表示
- オートコンプリート機能
- 表示順変更機能（新着順⇔お気に入り順）
- Ajax（Turbo Frames）
- フォローしているカテゴリーに新規テクニックが投稿されたことをメールでお知らせ
- パンくずリスト
- DB永続化
- RSpec（モデルテスト・システムテスト）
- CI（GithubActions）
- ファピコン
- 独自ドメイン
- レスポンシブ対応
